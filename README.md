## Money Manager System

 Features (Project is being updated)
- Add / update / delete, ... 
- Category management
- User profile support
- Monthly reports

Technologies for Backend
- Java Spring Boot
- MySQL
- Spring Data JPA,JWT,Cloudinary,OAuth2
- Brevo API
- Hibernate

How to run
1. Clone project
2. Configure database in `application.yaml`
3. Run `mvn spring-boot:run`
